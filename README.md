## 💡 주요 기능

- 반응형 디자인 : 테블릿(1024px), 모바일(390px)에서 최적화된 UI/UX 제공
- 오시는 길 : 위치 정보 및 지도 연동 기능
- 학습 테스트 : 유저정보 입력 후 영어 실력 진단 및 점수에 따른 반 배정 기능
- 프로그램 신청 : 다양한 프로그램 목록 확인
- 예약 페이지 : 달력에서 날짜 선택 후 프로그램 예약 기능
- 마을 소개 : 영어마을의 역사, 시설, 강사진 소개

## 📌 페이지 설명 
|                                    페이지                               |                                 설명                                    |
| :---------------------------------------------------------------------: | :---------------------------------------------------------------------: |
|                                메인페이지                                |  주요 사용자인 어린이와 어린 자녀를 둔 부모님을 위해 친근한 캐릭터를 활용한 디자인을 구성했습니다. 그리고 사이트의 목적에 맞게 가장 자주 사용하는 기능을 한눈에 확인할 수 있도록 직관적이고 편리한 UI/UX를 제공합니다.  |
|                                오시는 길                                | 영어마을의 위치 정보를 제공하며, 지도 연동 기능을 통해 쉽게 찾아올 수 있습니다.|
|                               학습 테스트                                | 사용자 정보 입력 후 영어 실력 진단 테스트를 진행할 수 있고, 20문제 중에 점수에 따라 적합한 반을 배정받을 수 있습니다.|
|                              프로그램 신청                                | 다양한 프로그램 목록을 확인이 가능하며, 4박5일 프로그램을 보면 예약하기라는 버튼이 있으며 예약 페이지로 이동이 가능합니다. |
|                               예약 페이지                                | 달력에서 원하는 날짜를 선택하여 프로그램을 예약할 수 있습니다. 프로그램 자체가 월요일에서 금요일까지 진행되는 프로그램이기 때문에 해당 날짜를 선택하면 그 주의 월요일에서 금요일까지 선택됩니다. 그리고 간단한 정보 입력 후 예약확인 모달 창이 뜨며, 예약 후 바로 학습 테스트로 반 배정을 받을 수 있게 연결되어 있습니다.|
|                                마을 소개                               |            영어마을의 역사, 시설, 강사진을 확인할 수 있습니다.                |
|                           모든 페이지 공통                               |     반응형 디자인을 통해 태블릿(1024px)과 모바일(390px)에서 최적화된 UI/UX를 제공합니다.        |

## ✅ 페이지 이동 경로<br />
|                       페이지                   |                                                      이동 방법                                                                           |
| :--------------------------------------------: | :------------------------------------------------------------------------------------------------------------------------------------: |
|               오시는 길 페이지                   |     header에 보이는 navigation에 영어마을소개 -> 오시는 길 클릭 / 메인페이지 가운데에 4가지 목록이 있는 부분에서 가장 오른쪽 오시는 길 클릭        |
|                    학습 테스트 페이지            |     header에 보이는 navigation에 프로그램 신청 -> 학습 테스트 클릭 / 메인페이지 가운데에 4가지 목록이 있는 부분에서 3번 째의 학습 테스트 클릭        |
|                 프로그램 신청 페이지             |     header에 보이는 navigation에 프로그램 신청 -> 프로그램 신청 클릭 / 메인페이지 가운데에 4가지 목록이 있는 부분에서 2번 째의 프로그램 신청 클릭    |
|                 예약 페이지                     |     프로그램 신청 페이지에 들어간 뒤 2번 째의 4박 5일 프로그램에 있는 예약하기 버튼 클릭        |
|              마을 소개 페이지                    |     header에 보이는 navigation에 영어마을소개 -> 마을 소개 클릭 / 메인페이지 오른쪽에 보이는 캐릭터 클릭 후 첫 번째로 있는 마을 소개 클릭        |
